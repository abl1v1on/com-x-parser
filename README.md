# Quick Guide
Создайте `.env` файл, скопируйте в него переменные из `.env-example` и замените их значения на ваш логин и пароль от аккаунта на https://com-x.life/.

> ⚠️ **Эти данные используются исключительно для получения необходимых для работы скрипта куков! Если боитесь за свой аккаунт - создайте новый и укажите его данные в `.env` файле**

Создайте виртуальное окружение и установите необходимые зависимости:
~~~shell
# win
python -m venv .venv
. .\.venv\Scripts\activate

# linux
python3 -m venv .venv
source .venv/bin/activate

# uv
uv sync
~~~

Запустите скрипт:
~~~shell
# win
python src/main.py

# linux
python3 src/main.py

# uv
uv run src/main.py
~~~

При первом запуске куки авторизации сохраняются в `cookies.json`. При последующих запусках появится возможность обновить их (например, если давно не пользовались скриптом):
~~~shell
Save cookies again? [Y/n]:
~~~

Далее указываем ссылку ведущую на страницу нужной нам манги:
~~~shell
Manga URL: https://com-x.life/11082-chelovek-benzopila-2-q2e5.html
~~~

> ⚠️ **После ввода ссылки на страницу манги начнется инициализация Chrome драйвера для работы Selenium. Если вы из России, то для его установки может потребоваться VPN. Включите его перед запуском скрипта.**

Вы также можете собрать дополнительные данные о манге (эти данные сохраняются в директории `metadata` в корне проекта.), такие как:

- Обложка
- Описание
- Теги

Далее начнется установка всех глав в директорию `downloads`, которая расположена в корне проекта. После окончания установки вы увидите соответствующее сообщение:
~~~shell
Установлено 100 из 100 файлов!
~~~

- Добавить возможность выбирать директорию для установки глав и метаданных;
- Сбор метаданных и идентификаторов глав в headless моде;
- Возможность доустановки недостающих глав при возникновении ошибок в ходе установки;
- Возможность выбирать браузеры помимо Chrome;
