# Quick Guide
Склонируйте репозиторий и перейдите в появившеюся директорию:
~~~shell
git clone https://github.com/abl1v1on/com-x-parser.git
cd com-x-parser
~~~

Создайте виртуальное окружение и установите необходимые зависимости:
~~~shell
# win
python -m venv .venv
. .\.venv\Scripts\activate
pip install -r requirements.txt

# linux
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# uv
uv sync
~~~

Запустите скрипт:
~~~shell
# win
python src/main.py

# linux
python3 src/main.py

# uv
uv run src/main.py
~~~

При первом запуске скрипт попросить ввести ваш логин и пароль на сайте https://com-x.life/. Во время ввода пароля его не будет видно. Данные можно будет изменить вручную в файле `user-config.json`.

> ⚠️ **Эти данные используются исключительно для получения необходимых для работы скрипта куков! Если боитесь за свой аккаунт - создайте новый.**

 При последующих запусках вводить логин и пароль не нужно, а также появится возможность обновить куки (например, если давно не пользовались скриптом):
~~~shell
Save cookies again? [Y/n]:
~~~

Далее указываем ссылку ведущую на страницу нужной нам манги:
~~~shell
Manga URL: https://com-x.life/11082-chelovek-benzopila-2-q2e5.html
~~~

> ⚠️ **После ввода ссылки на страницу манги начнется инициализация Chrome драйвера для работы Selenium. Если вы из России, то для его установки может потребоваться VPN. Включите его перед запуском скрипта.**

Вы также можете собрать дополнительные данные о манге (эти данные сохраняются в директории `metadata` в корне проекта.), такие как:

- Название
- Описание
- Обложка
- Теги

Далее начнется установка всех глав в директорию `downloads`, которая расположена в корне проекта. После окончания установки вы увидите соответствующее сообщение:
~~~shell
Установлено 100 из 100 файлов!
~~~
